# Backend Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build and containerise the FastAPI backend service — a minimal Cloud Run service with three endpoints: health check, graph JSON proxy, and contact form.

**Architecture:** Single FastAPI app with two routers. The graph router fetches `graph.json` from GCS and returns it (keeps the bucket private, adds caching headers). The contact router validates input and sends email via the Resend API. All tests use `pytest` + `httpx.AsyncClient` with dependency injection to swap out real GCS/Resend calls.

**Tech Stack:** Python 3.13, FastAPI, uv, pytest, httpx, google-cloud-storage, resend, slowapi

**Prerequisites:** infra plan complete (Secret Manager slots exist, GCS bucket created)

```python
import json
import pytest
from unittest.mock import AsyncMock, patch
from httpx import AsyncClient, ASGITransport
from app.main import app
from app import core


SAMPLE_GRAPH = {
    "nodes": [{"id": "skill-python", "type": "skill", "label": "Python"}],
    "edges": [],
}
```

- [ ] **Step 3: Create `backend/app/core/gcs.py`**

```python
from google.cloud import storage


async def fetch_object(bucket_name: str, key: str) -> bytes:
    """Fetch a GCS object and return its contents as bytes."""
    client = storage.Client()
    bucket = client.bucket(bucket_name)
    blob = bucket.blob(key)
    return blob.download_as_bytes()
```

- [ ] **Step 4: Implement `backend/app/routers/graph.py`**

```python
import json
from fastapi import APIRouter, HTTPException
from fastapi.responses import JSONResponse

from app import core
from app.config import settings

router = APIRouter()


@router.get("/graph")
async def get_graph() -> JSONResponse:
    try:
        data = await core.gcs.fetch_object(settings.gcs_bucket, "graph.json")
    except Exception:
        raise HTTPException(status_code=502, detail="Graph data unavailable")

    return JSONResponse(
        content=json.loads(data),
        headers={"Cache-Control": "public, max-age=300"},
    )
```

```python
import resend
from fastapi import APIRouter
from pydantic import BaseModel, EmailStr

from app.config import settings

router = APIRouter()


class ContactRequest(BaseModel):
    name: str
    email: EmailStr
    message: str


@router.post("/contact")
async def contact(body: ContactRequest) -> dict:
    resend.api_key = settings.resend_api_key
    resend.Emails.send({
        "from": "site@tapshalkar.com",
        "to": "aditya@tapshalkar.com",
        "subject": f"Portfolio contact from {body.name}",
        "text": f"From: {body.name} <{body.email}>\n\n{body.message}",
    })
    return {"ok": True}
```
