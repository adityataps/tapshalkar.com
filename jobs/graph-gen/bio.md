# About Aditya Tapshalkar

I'm a software engineer focused on machine learning, AI systems, and the products built on top of them. I'm based in Atlanta, GA.

## Education

B.S. Computer Science, Georgia Institute of Technology (Georgia Tech). Coursework in machine learning, algorithms, systems programming, and data structures.

## Experience

Most of my career has been at the intersection of applied ML and product engineering — building systems that are both technically rigorous and actually useful to people. I've worked across early-stage startups and larger engineering teams, primarily as a Machine Learning Engineer and backend engineer.

My work has spanned LLM fine-tuning and evaluation pipelines, real-time inference infrastructure, knowledge graph construction, and API design. I care a lot about the full stack — from the model and data pipeline down to the interface the user actually touches.

## Projects

**tapshalkar.com** — This portfolio site. Monorepo with a Next.js static frontend (GCS + Cloud CDN), FastAPI backend on Cloud Run, and a daily Cloud Run Job that fetches live data from GitHub, Spotify, Steam, Trakt, and Apple Health, then calls Claude to synthesize a typed knowledge graph. Features an agentic chatbot ("digital me") powered by Claude with tool use and graph-grounded responses.

Other projects are on GitHub — spanning ML experiments, developer tooling, and full-stack applications.

## Values

I prefer direct communication, working in small focused teams, and shipping things. I'm skeptical of complexity for its own sake and try to find the simplest solution that actually solves the problem. I think good engineering judgment is about knowing when not to build something as much as knowing how.
